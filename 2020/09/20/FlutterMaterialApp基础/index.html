常见部件

font部件

java

使用Text部件，可以在里面设置多种text的style，设置fontSize,fontWeight,color, fontStyle等等。
在fontWeight中，可以使用FontWeigh返回一个字重;
在color中，使用Colors返回颜色，
在fontStyle中，使用FontStyle中的斜体格式。

container 组件

Container 在没有child的时候，用parent的大小，在有child时就会去fit它的child，如果有设置长宽的时候，就会fit我们设置的长宽

java

card

设计说明：
https://material.io/components/cards

card中不可嵌套card，但是可以放别的物件

textField 输入框

通过decoration属性进行修饰，通过onChanged等handle event进行操作。也可以通过controller获取输入的数据
