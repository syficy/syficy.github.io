centos 使用docker 安装 anaconda+jupyter notebook，通过web访问
Created 2020-01-03| Updated 2020-01-21|操作系统centos
Post View:

查找anacond的镜像

docker search anaconda

拉取镜像

docker pull continuumio/anaconda3

运行容器

docker run -t -i continuumio/anaconda3 /bin/bash

进入内部的bash
注意此时bash中@的设备已经不一样了
