Linux硬盘管理

硬盘结构

储存设备，包括光盘，硬盘，U盘，网络SAN，NAS等，其中最主要的是硬盘，分为机械硬盘（Hard Disk Drive) 和 固态硬盘(Solid State DIsk),前者通过磁性碟片存储数据，后者通过闪存克里存储。

机械硬盘

如下图，HDD由512byte的扇区围成的环形同心圆磁道组成，最外侧的为0磁道，向内递增。
上下多个盘片形成柱面。硬盘大小为 磁头数 x 柱面数 x 扇区数 x 每个扇区的大小，常见接口有IDE\SATA\SCSI。

固态硬盘 SSD

采用闪存颗粒进行存储，比较流行的是NVEM协议，共四种长度尺寸，寿命有限。

linux文件系统

不同的操作系统有不同的文件系统， windows98以前： FAT16，2000之后：NTFS；早期Linux：Ext2，现在linux：Ext4，格式化相当于给硬盘写入相应的文件系统。文件系统格式化出inode和block块，inode中存储文件权限和属性，block存储真实数据，这种系统也成为索引式文件系统

linux识别硬盘

系统开启时，linux会根据MBR（Master Boot Record）识别硬盘系统，包含前446个boot loader程序和后64个分区表信息。
