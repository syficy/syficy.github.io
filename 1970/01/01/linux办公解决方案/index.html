linux 办公解决方案

linux作为使得开发者舒服无比的操作系统，在办公应用上尚未有windows那样完善的生态，但是总有开发者维护着一批可以办公的项目，此贴用于收集此类项目。

音乐

项目

中文音乐 music-dl
https://github.com/0xHJK/music-dl

安装

安装方式极为简单，推荐pip安装
pip3 install pymusic-dl

使用

使用方式极为简单，命令行输入 music-dl,即可调用

音视频剪辑

项目

FFmpeg
https://github.com/FFmpeg/FFmpeg

安装

官网
https://www.ffmpeg.org/download.html

使用

ryf的博客用中文介绍了几个简单的用法，包括转码，裁剪，提取音频
http://www.ruanyifeng.com/blog/2020/01/ffmpeg.html

学习强国

项目

fuckxuexiqiangguo
https://github.com/fuck-xuexiqiangguo/Fuck-XueXiQiangGuo

安装

直接下载编译过的版本就行

使用

扫码登入，偶尔出现无法刷分的现象，需要手去多动动

qq

项目

qq 应付国产操作系统版
https://im.qq.com/linuxqq/index.html

安装

下载编译过的版本就行

使用

扫码登入，经常闪退，不到万不得已不要用

思维导图

项目

xmind
https://www.xmind.net/

安装

下载编译过的版本就行

使用

跟windows一样

命令行代理

项目

proxychains4
https://github.com/rofl0r/proxychains-ng

安装

见文档

使用

proxychains4 + 命令 即可
